# Groundwork Marketplace

A Claude Code marketplace for discovering and installing the Groundwork plugin.

## Adding this Marketplace

```bash
claude plugin marketplace add https://github.com/etr/groundwork-marketplace
```

## Installing Groundwork

Once the marketplace is added, install the Groundwork plugin:

```bash
claude plugin add groundwork
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **groundwork** | Comprehensive skills library: planning, design, TDD, debugging, collaboration patterns, and proven techniques. 31 skills, 7 agents, 11 commands. |

## Direct Installation

You can also install Groundwork directly without using the marketplace:

```bash
claude plugin add https://github.com/etr/groundwork
```
